Utilitários de Exibição

Para criar um elemento de bloco, adicione a classe d-block. Use quaisquer classes d-*-block para controlar QUANDO o elemento deve ser um elemento de bloco em um comprimento específico de tela.

Ajuste o tamanho da janela do browser para ver o efeito.

d-block d-sm-block d-md-block d-lg-block d-xl-block
