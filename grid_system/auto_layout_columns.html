Colunas de Layout Automático

No Bootstrap 4, há uma maneira fácil de criar colunas de comprimento igual: apenas use a classe .col-size em um especificado número de elementos. O Bootstrap irá reconhecer quantas colunas estão ali e cada coluna terá o mesmo comprimento.

Duas colunas: 50% de comprimento em todas as telas, exceto para as extra pequenas (100% de comprimento em telas com menos de 576px de comprimento)

2 de 2

Quatro colunas: 25% de comprimento em todas as telas, exceto para as extra pequenas (100% de comprimento em telas com menos de 576px de comprimento)
