Fundos Contextuais

Use classes de fundo contextuais para fornecer "significado através das cores".

Note que você também pode adicionar uma classe .text-* se você desejar uma cor de texto diferente:

Este texto é importante.

Este texto indica sucesso.

Este texto representa alguma informação.

Este texto representa um aviso.

Este texto representa perigo.

Cor de fundo secundária

Cor de fundo cinza escuro.

Cor de fundo cinza claro.
